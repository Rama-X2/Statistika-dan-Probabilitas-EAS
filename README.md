# Statistika dan Probabilitas - EAS

Web aplikasi untuk perhitungan statistika dan probabilitas yang mencakup:
- Analisis Regresi Linier Sederhana
- Kalkulator Rumus Slovin

- **Nama:** Ade Ramadhani Putra

## Fitur

### 1. Analisis Regresi Linier
- Input data concurrent users vs response time
- Perhitungan otomatis persamaan regresi
- Visualisasi grafik scatter plot dengan garis regresi
- Prediksi kapasitas maksimal server
- Export hasil perhitungan

### 2. Kalkulator Rumus Slovin
- Perhitungan ukuran sampel penelitian
- Berbagai tingkat margin of error
- Visualisasi perbandingan populasi vs sampel
- Langkah perhitungan detail
- Interpretasi dan rekomendasi

## Teknologi
- HTML5
- Tailwind CSS
- Chart.js
- Lucide Icons
